# Pattern Demos

Konkrete Code-Beispiele für ausgewählte Patterns aus der AI Agent Pattern Landscape.

## Demos

- `react_tool_calling.py`: ReAct und Tool Calling mit LangChain
- `sequential.py`: Sequential Pipeline mit LangChain Prompt Chains
- `routing.py`: Routing mit LangGraph Conditional Edges
- `reflection.py`: Evaluator-Optimizer und Reflexion als LangGraph Loop
- `state_graph.py`: Supervisor und Handoff als State Graph
- `memory.py`: Memory und Checkpointing
- `deepagents_demo.py`: Deep Agents mit Coordinator, Tools und Subagents

## Ausführen

```bash
agent-patterns list
agent-patterns run all
agent-patterns run deepagents "Research how handoffs should be documented."
```
