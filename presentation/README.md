# Presentation

Reveal.js Präsentation für Talks, Workshops und Veröffentlichungen zur AI Agent Pattern Landscape.

## Start

```bash
npm install
npm run dev
```

Der lokale Dev-Server wird von Vite gestartet. Die Slides liegen in `src/main.js`, das Styling in `src/styles.css`.

## Inhalte

- `index.html`: Einstiegspunkt für Reveal.js
- `src/`: Slide-Inhalt und Präsentationsstil
- `assets/`: verlinkte Medien, insbesondere die Pattern-Landscape-Infografik

## Stil

Die Präsentation orientiert sich am Workshop-Stil aus `workshops/003-ai-agents-langgraph`: dunkler Hintergrund, klare Typografie, kontrastreiche Pattern-Karten und keine Emojis in den Slides.
