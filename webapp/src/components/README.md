# Components

Featureübergreifende Atomic-Design-Komponenten der Webapp.

## Struktur

- `atoms/`: kleinste UI-Bausteine ohne Fachlogik
- `molecules/`: zusammengesetzte UI-Elemente
- `organisms/`: komplexe, wiederverwendbare UI-Bereiche

Feature-spezifische Komponenten gehören in `../features/<feature>/`.
